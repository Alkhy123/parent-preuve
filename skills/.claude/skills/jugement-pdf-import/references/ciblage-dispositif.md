# Ciblage du dispositif

## Priorité d’extraction

Chercher dans cet ordre :

1. Section commençant par “PAR CES MOTIFS”
2. Section contenant des verbes de dispositif :
   - DIT
   - FIXE
   - CONDAMNE
   - ORDONNE
   - CONSTATE
   - RAPPELLE
   - DÉBOUTE
3. Fin du jugement si le titre du dispositif est absent.
4. Texte complet seulement en dernier recours.

## Règle anti-erreur

Ne jamais préférer une demande des parties à la décision finale.

Exemple :

- Motif : “Madame demande 300 euros.”
- Dispositif : “Fixe la contribution à 180 euros.”

La valeur extraite doit être 180 euros.
