# Anti-inférence Parent Preuve

L’IA ne doit pas déduire :

- intention
- mauvaise foi
- caractère volontaire
- motif d’un DVH médiatisé
- dangerosité
- diagnostic psychologique
- infraction
- appel
- exécution provisoire
- enfant_id

Si le texte ne dit pas une information, renvoyer `null`.

Si le texte est ambigu, confiance `moyenne` + avertissement.
